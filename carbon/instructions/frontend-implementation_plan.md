# CARBON APPLICATION  
## FRONTEND ARCHITECTURE DOCUMENTATION (PRODUCTION-GRADE)

---

# 1. OVERVIEW

Carbon is a **parametric insurance mobile application** designed to provide **automated financial protection** for users (primarily gig workers such as delivery partners) against **income disruptions caused by external events**.

Unlike traditional insurance systems:

- No manual claim filing is required  
- Claims are **automatically triggered by events**  
- Payouts are processed without user intervention  

The frontend application acts as:

> A **real-time visibility layer** over an automated insurance engine
